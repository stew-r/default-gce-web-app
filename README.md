# GCE Demo Web App

This is a small demo app that can be containerized and deployed to Google
Compute Engine.

## Usage instructions

Get the repo:

```bash
git clone https://github.com/stew-r/gce-demo-web-app.git`
```

Navigate to the directory:

```bash
cd gce-demo-web-app
```

Run the app:

```bash
sh cmd/run.sh
```

Send a request to the app:

```bash
curl localhost:80
```
